# Exercise 2: E-commerce Platform Search Function

## Objective
Implement Linear Search and Binary Search for an e-commerce platform.

## Theory

### What is Big O Notation?
Big O notation measures how an algorithm's running time grows as the input size increases.

### Time Complexities

| Algorithm | Best | Average | Worst |
|-----------|------|---------|-------|
| Linear Search | O(1) | O(n) | O(n) |
| Binary Search | O(1) | O(log n) | O(log n) |

## Why Binary Search?
Binary Search is suitable for large sorted datasets because it repeatedly halves the search space.
